# SlackPost

create your apps.  
<https://api.slack.com/apps  >
  
1. [Create New App]  
  - **App Name** : any value.
  - **Development Slack Workspace** : your workspace selected  
2. [Create App]  

3. [Permissions]  

4. Scopes  
   - search:read  
   - channels:read  
   - groups:read  
   - chat:write:user  

5. [Save Changes]

6. [Install App to Workspace]

7. [Authorize]
